# book-collection
Online book collection with customizable book info

## Motivation
>"Buying books and reading books are two different hobbies"
>
I enjoy collecting pretty books and sometimes it happens that I have multiple editions of the same book. I've tried tracking my collection on Goodreads, but it seems that it doesn't care about the editions. So I've tried moving to Notion. But after about 100 books, I gave up on copy-pasting all the info I wanted available manually. That's when I started this project.

## Intended Functionality
The user inputs a book's ISBN. The app attempts to pull online info into the database. Then the user can edit non-objective info to their liking. The database will hold all the books the user inputs.

## Information stored about a book
### "Objective":
- ISBN
- Title
- Subtitle
- Authors (coauthors, illustrators, translators, editors...)
- Publisher
- Imprint
- First publication date
- This edition's publication date
- Language
- Number of pages
- Weight (in grams)
- Width (in mm)
- Height (in mm)
- Depth (in mm)

### "Subjective":
- Image to be displayed
- Ownership status (own, owned, borrowed, want...)
- Reading status (read, want to read, currently reading...)
- Collection (e.g. Barnes & Noble Pocket Leather Editions, Wordsworth Collector's...)
- Series (e.g. Sherlock Holmes, The Lord of the Rings...)
- Format (hardcover, softbound, e-book, audiobook...)
- Country of the setting
- Era of the setting (1800's, 19's, medieval...)
- Genres
